# Jenkins Blueocean

Template docker Jenkins

### Iniciando Docker
```console
docker-compose up -d
```

### Pegando chave do jenkins
```console
docker exec jenkins-blueocean cat /var/jenkins_home/secrets/initialAdminPassword
```
